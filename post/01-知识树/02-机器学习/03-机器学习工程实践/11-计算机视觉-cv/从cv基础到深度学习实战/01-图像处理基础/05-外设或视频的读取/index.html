Video 的读

VideoCapture cap(0);//指向的是设备的编号

以前用的时候，一直想知道设备的编号怎么得到？比如说我有两个摄像头，安装的顺序不同，编号也不同，我想指定其中一个摄像头怎么做到？之前是通过摄像头的名称来查找对应的编号，但是之前使用的不是 opencv 自带的摄像头检测的，而是使用的 qt 的。好像没找到在 opencv 中这个怎么做？

VideoCapture cap(“video.short.raw.avi”);
if(!cap.isOpened())
{
    cerr << “Can not open a camera or file.” << endl;
    return -1;
}
Mat edges;
namedWindow(“edges”,1);
for(;;)
{
    Mat frame;
    cap >> frame;
    if(frame.empty())
        break;
    ….
    if(waitKey(30) >= 0)
        break;
}

关于编码的 video 格式，我们看一下这个网站：http://www.fourcc.org/codecs.php 什么意思？什么编码？

老师这个地方说了他遇到的一个问题：在双目多目的时候，有的时候会出现摄像头打不开的情况，其实这个时候是硬件的问题，注意你硬件的 id，如果你的两个设备是通过 hub 连接的，那么他们的 id 可能是一个。这个还真的不确定以前是不是遇到这个问题，以前遇到的应该不是这个问题。

还有一个问题：我们经常使用 opencv 打开 avi 格式文件，有时候可以打开，有时候不行，这个主要是由 avi 文件的压缩算法不同导致的，要看下是否要把你的源文件重新编码成 avi 文件来读取。这个的确是遇到过这个问题的。

Video 的写

Size s(320, 240);
VideoWriter writer = VideoWriter(“myvideo.avi”,CV_FOURCC('M','J','P','G'), 25, s);
if(!writer.isOpened())
{
    cerr << “Can not create video file.\n” << endl;
    return -1;
}
//视频帧
Mat frame(s, CV_8UC3);
for(int i = 0; i < 100; i++)
{
    writer << frame;
}
