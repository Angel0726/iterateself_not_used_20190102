拆分及合并图像通道

有时我们需要对 BGR 三个通道分别进行操作。这是你就需要把 BGR 拆分成单个通道。嗯，一般什么时候使用这种方式呢？

有两种方法：

  • cv2.split() 对应 cv2.merge()
  • img[:,:,2]

cv2.split() 是一个比较耗时的操作，只有真正需要时才用它，能用 Numpy 索引就尽量用。嗯，之前不知道，好的。

下面看一个例子：

import cv2
import numpy as np

img = cv2.imread('./1.jpg')
cv2.imshow('origin', img)

b, g, r = cv2.split(img)
cv2.imshow('red_1', r)
r2 = img[:, :, 2]
cv2.imshow('red_2', r2)

img = cv2.merge([b, g, r])
cv2.imshow('merge_img', img)
img = cv2.merge([b, g, g])
cv2.imshow('merge_b_g_g', img)
cv2.waitKey(-1)

显示如下：

origin:

red_1:

red_2:

merge_img:

merge_b_g_g:

假如你想使所有像素的红色通道值都为 0，你不必先拆分再赋值。你可以 直接使用 Numpy 索引，这会更快。

import cv2
import numpy as np

img = cv2.imread('./1.jpg')
cv2.imshow('origin', img)

img[:, :, 2] = 0
cv2.imshow('red_zero', img)
cv2.waitKey(-1)

输出如下：

origin:

red_zero:
