#### Q 6* :<span style="color:red;">这题没看懂</span>

题目：
32位机上根据下面的代码，问哪些说法是正确的？

```
signed char a = 0xe0;
unsigned int b = a;
unsigned char c = a;
```

答案:
b的十六进制表示是：0xffffffe0

解答:
考察有符号数和无符号数之间的转换。
1. a : 1110 0000。
2. 扩展问题：
    长 -> 短：低位对齐，按位复制。
    短 -> 长：符号位扩展。
3. 精度提升：
    两个变量运算，表示范围小的变量精度达的变量提升（signed -> unsigned）。
